# Skylight Feature Audit & Roadmap

## Skylight Calendar 2 (2026) - Core Marketed Features

Based on research from CES 2026 announcements and official Skylight website:

### 1. **Unified Family Calendar** ✅ We Have This
- Syncs from Google Calendar, iCal, Microsoft Calendar, TeamSnap
- Color-codes events by family member
- Multiple view modes: day, week, month, schedule
- **Lumina Status**: ✅ Implemented with Google Calendar sync, ready for iCloud

### 2. **AI Photo Recognition & Event Import** ❌ Missing
- Photograph paper flyers/schedules
- AI extracts events automatically and adds to calendar
- **Lumina Status**: ❌ Not implemented (future enhancement)

### 3. **Tasks Manager / Chores System** ⚠️ Unique Addition (Planned)
- Interactive checklists for daily routines
- Color-coded task assignment per family member
- Encourages kids' independence
- **Lumina Status**: ⚠️ Planned (similar to HomeGlow Chores widget)

### 4. **Meal Planning & Recipe Integration** ❌ Missing
- Recipe suggestions
- Auto-generates ingredient lists
- Instacart integration for ordering
- **Lumina Status**: ❌ Not implemented

### 5. **Grocery Lists & Shopping** ❌ Missing
- Shared family grocery lists
- Direct integration with delivery services
- **Lumina Status**: ❌ Not implemented

### 6. **Photo Display When Idle** ❌ Missing
- Shows family photos when calendar not in active use
- Digital picture frame mode
- **Lumina Status**: ❌ Not implemented (planned for kiosk mode)

### 7. **Weather Forecasting** ❌ Placeholder Only
- Location and time-specific weather for events
- **Lumina Status**: ⚠️ Placeholder in UI (not live data)

### 8. **Appointment Reminders** ⚠️ Partial
- Notifications for upcoming events
- **Lumina Status**: ⚠️ Events visible, no push notifications yet

## Lumina's Unique Features

### 1. **WooCommerce Order Dashboard** ✅ Unique to Lumina
- Real-time order sync and status tracking
- Order revenue statistics
- Today's orders priority view
- Native calendar integration (orders as events)
- **Status**: ✅ Fully implemented

### 2. **Multi-Business Management** ✅ Unique to Lumina
- Family-centric architecture with business integration
- Separate family/personal/business event types
- **Status**: ✅ Architecture in place

## Competitive Advantage

Lumina's unique position:
- **Business Integration**: WooCommerce orders as first-class calendar events
- **Open Source**: Self-hosted, no subscription required
- **Configurable**: Web-based settings panel (vs. Skylight's fixed config)
- **Developer-Friendly**: Docker deployment, API-first architecture
